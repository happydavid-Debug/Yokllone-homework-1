作业发布器 API 使用示例

1. 获取今日作业

获取当前日期的作业内容

2. 创建/更新作业

为指定日期创建或更新作业内容

3. 获取作业列表

获取日期范围内的作业列表

4. 删除作业

删除指定日期的作业内容

代码示例

以下是使用JavaScript调用API的示例代码：

// 创建API客户端实例 const api = new AssignmentAPI(); // 获取今日作业 const today = AssignmentAPI.formatDate(new Date()); const result = await api.getAssignment(today); // 更新作业 await api.updateAssignment(today, '作业内容...'); // 获取作业列表 const list = await api.getAssignments({ start: '2025-12-01', end: '2025-12-31', limit: 10 });
